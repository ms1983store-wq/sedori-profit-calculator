# 仕入れノート

せどりの仕入れ現場で、商品を店舗ごとに記録し、その日の店舗数・仕入れ額・想定粗利を確認するためのローカルWebアプリです。

## できること

- 仕入れ日、店舗、商品、数量、仕入れ単価、想定売価を記録
- 販売手数料と送料・梱包費を含む想定粗利を自動計算
- 日ごとの仕入れ店舗数、点数、仕入れ総額、想定売上、想定粗利、ROIを集計
- 店舗ごとの想定粗利で絞り込み
- 商品の編集・削除、商品名・店舗名・JAN/ASIN・メモの検索
- 全記録のCSV保存と読み込み
- CSV内容の貼り付け読み込み
- `2026せどり管理ツールB_ver2.2 - 管理表.csv` の6段見出しを自動判別して取り込み
- 売却済みは実手数料・売上日・販売ルートを保持し、在庫の売価未入力分は損益分岐価格を仮設定
- 管理Noによる再読み込み時の重複防止
- ブラウザのローカルストレージへの自動保存
- スマートフォン表示、ホーム画面への追加、オフライン利用に対応

## 想定粗利の計算

```text
販売手数料（1点） = 想定売価 × 販売手数料率（1円未満切り上げ）
想定粗利 =（想定売価 − 仕入れ単価 − 販売手数料 − 送料・梱包費）× 数量
ROI = 想定粗利 ÷ 仕入れ総額 × 100
```

## 起動方法

PowerShellでこのフォルダを開き、次を実行します。

```powershell
.\serve.ps1
```

表示されたURLをブラウザで開いてください。データは利用中のブラウザ内だけに保存されるため、端末移行やバックアップには「CSV保存」を使います。

## ファイル構成

- `index.html` — 画面
- `styles.css` — レイアウトとデザイン
- `app.js` — 保存、集計、CSV入出力
- `manifest.webmanifest` / `sw.js` — PWA・オフライン対応
