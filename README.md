# せどり在庫帳

せどり商品の仕入れ、販売日、販売価格、損益分岐点、利益をまとめて管理する静的 Web アプリです。Seller Book のように取引状態を一覧管理しながら、商品ごとに「販売日」と「損益分岐点（販売価格）」を確認できます。

## 主な機能

- 商品名、販売先、状態、仕入日、販売日、仕入れ値、販売価格、送料、梱包費、手数料率、メモを登録
- 状態は `在庫`、`売却済み`、`出品前`、`出品中`、`発送準備`、`評価待ち`、`完了` に対応
- 仕入れ値、送料、梱包費、手数料率から損益分岐点となる販売価格を自動計算
- 販売価格を入力すると利益と利益率を自動計算
- 在庫数、在庫原価、今月売上、今月利益を集計
- 商品名やメモ検索、状態フィルター、CSV 書き出し・読み込みに対応
- `2026せどり管理ツールB_ver2.2 - 管理表.csv` 形式の管理表CSVを在庫一覧へ取り込み可能
- CSV貼付での読み込みにも対応
- データはブラウザのローカルストレージに保存

## 計算式

- 販売手数料 = 販売価格 × 手数料率（1円未満切り上げ）
- 管理表CSVから手数料実額が取り込まれた売却済み商品は、その実額を優先
- 総コスト = 仕入れ値 + 送料 + 梱包費
- 利益 = 販売価格 - 販売手数料 - 総コスト
- 利益率 = 利益 ÷ 販売価格 × 100
- 損益分岐点 = 手数料差し引き後に総コストを回収できる最小販売価格

## 公開

`index.html`、`styles.css`、`app.js`、`manifest.webmanifest`、`sw.js`、`icons/` をそのまま静的ホスティングに置くと公開できます。
GitHub Pages、Cloudflare Pages、Netlify などの常時公開ホスティングに対応しています。
